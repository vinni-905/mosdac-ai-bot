About This Project

The Problem

ISRO's MOSDAC portal is a national treasure, containing vast amounts of satellite data crucial for research, disaster management, and education. However, its complex structure can be challenging for both new and experienced users to navigate, making it difficult to find specific information quickly.

Our Solution

The MOSDAC AI Portal is a hackathon project designed to solve this problem. We've built an intelligent layer on top of the existing portal. By leveraging modern AI technologies like Natural Language Processing (NLP) and Retrieval-Augmented Generation (RAG), we provide a simple, conversational interface to access this complex data.

Our AI assistant understands user questions, searches a curated knowledge base of MOSDAC information, and provides direct, accurate answers, saving users valuable time and effort.
